# Cómo elegir la impresora de etiquetas industrial correcta

Elegir una impresora de etiquetas no es solo cuestión de precio: el volumen diario, el ambiente operativo y el material de la etiqueta dictan la decisión técnica. Aquí desglosamos los criterios clave.

## 1. Térmica directa vs. transferencia térmica

La **térmica directa** usa papel sensible al calor; no requiere ribbon, pero la imagen se degrada con calor o luz UV. Ideal para etiquetas de corta vida (envíos, recibos).

La **transferencia térmica** usa un ribbon de cera, cera-resina o resina pura. Produce etiquetas duraderas resistentes a químicos, abrasión y temperaturas extremas — la elección para activos fijos, industriales y exteriores.

## 2. Resolución: 203, 300 o 600 dpi

- **203 dpi** — texto y códigos de barras estándar
- **300 dpi** — texto pequeño, códigos densos, logos
- **600 dpi** — etiquetas miniatura (electrónica, joyería)

## 3. Ancho de impresión

Las impresoras industriales típicas manejan 4" de ancho. Para palets y etiquetas grandes, considera 6" u 8".

## 4. Volumen diario

| Volumen           | Tipo recomendado               |
| ----------------- | ------------------------------ |
| < 500 etiq/día    | Escritorio (ZD421)             |
| 500–4000 etiq/día | Industrial compacta            |
| > 4000 etiq/día   | Industrial de alto rendimiento |

## 5. Conectividad

USB para uso aislado, Ethernet para integración a ERP, WiFi para flotas móviles. Confirma que tu impresora soporte el protocolo que usa tu WMS o ERP.
